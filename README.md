# Kulindağ Dağ Evi - Web Sitesi Yenileme Projesi

Bu proje, Kulindağ Dağ Evi için modern, responsive ve estetik bir landing page tasarımıdır.

## Teknolojiler

- **Next.js 15** (App Router)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animasyonlar)
- **Lucide React** (İkonlar)
- **TypeScript**

## Kurulum

Proje dosyalarını indirdikten sonra:

```bash
npm install
npm run dev
```

## Özellikler

- **Modern Tasarım:** Rustic Minimalism ve Modern Cabin konsepti.
- **Responsive:** Mobil ve masaüstü uyumlu.
- **Animasyonlar:** Yumuşak geçişler ve scroll efektleri.
- **Bento Grid Galeri:** Modern görsel sunumu.
